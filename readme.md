## 开始

您可以 `Fork` 本仓库作为参考并开始自己的创意。



每个项目中至少有四个文件：

- `index-START.html` : 您需要完善的文件。该文件为您提供了一些基础代码，您需要完善它。
- `index-FINISHED.html`: 最终效果。您需要参考它，但不要完全模仿，建议多一点创新。
- `index-ME.html`: 我模仿 `index-FINISHED.html` 做的页面。
- `README.md`: 我对该项目的总结。

另外有些项目可能会有一些资源文件（例如一些css，js以及一些图片和音乐）。
